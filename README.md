# Internet & Web Tech. Final Project

## Project Overview

- Web-based application designed to analyze personal video game preferences against industry critical consensus. 
- The application fetches Metacritic scores for game titles between personal ratings and those of professional critics.

## Features

- Data retrieval from the RAWG Video Games Database.
- Calculation of rating divergence.
- Gaming taste based on statistical alignment with critics.
